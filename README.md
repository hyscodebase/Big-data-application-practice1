# 📊 Big Data Application Practice 1 (빅데이터 활용 실습 1)

## 📝 프로젝트 소개 (Description)
2026학년도 1학기 **'빅데이터 활용 실습 1'** 강의를 수강하며 학습한 내용과 실습 코드를 주차별로 아카이빙하는 레포지토리입니다. 
데이터 수집부터 전처리, 탐색적 데이터 분석(EDA), 데이터 시각화, 텍스트 마이닝, 그리고 머신러닝 기초 모델링까지 데이터 분석의 전 과정을 다룹니다.

## 🛠️ 기술 스택 (Tech Stack)
- **Language:** Python
- **Data Analysis & Preprocessing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Machine Learning:** Scikit-learn
- **Environment:** Jupyter Notebook, Google Colab

## 🚀 실행 방법 (How to Run)
1. 레포지토리를 클론합니다.
